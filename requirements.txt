# ── Platform notes ────────────────────────────────────────────────────────────
#
# macOS
#   tkinter: bundled with the python.org installer.
#            Homebrew Python: brew install python-tk
#
#   slcan (CANable 1.x / slcan firmware) — no driver needed on macOS 11+
#     Find your device:  ls /dev/tty.usb* /dev/cu.usb*
#
#   gs_usb (CANable 2.0 / candleLight firmware) — no driver needed
#     pip install pyusb libusb1
#
#   PCAN-USB: download PEAK macOS driver from peak-system.com
#
# Linux / Raspberry Pi
#   SocketCAN is in the kernel; no extra package needed.
#   Bring up:  sudo ip link set can0 up type can bitrate 500000
#   tkinter:   sudo apt install python3-tk
#
# Windows
#   slcan: pyserial is already listed above
#   PCAN: PEAK Windows driver from peak-system.com
